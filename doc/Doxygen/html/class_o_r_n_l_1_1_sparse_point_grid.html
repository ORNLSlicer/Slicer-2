Detailed Description

template<class T, class Locator>
class ORNL::SparsePointGrid< T, Locator >

Sparse grid which can locate spatially nearby elements efficiently.

Template Parameters
T- The element type to store
Locator= The functor to get the location from T. Locator must have: Point operator()(const T &t) const which returns the location associated with val.
