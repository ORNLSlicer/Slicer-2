Detailed Description

Namespace meant to encapsulate algorithms for determining which triangles of a mesh intersect with a mouse ray.

This namespace is meant to be used with an OpenGL view. To perform its operations it is necessary to have transformation matrices, so a view matrix is passed into certain methods when required.
