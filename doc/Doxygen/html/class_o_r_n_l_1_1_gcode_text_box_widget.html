◆ getCursorBlockNumber()

int ORNL::GcodeTextBoxWidget::getCursorBlockNumber ( )

This function is a convenience funciton that returns the block number in which the cursor is currently highlighted on.

Note
The block numbers are 0 indexed, so line number 1 corresponds to block number 0.
This function takes into account that QPlainTextEdit has each block as its own line.
Returns
Block number that cursor is currently highlighted on.
