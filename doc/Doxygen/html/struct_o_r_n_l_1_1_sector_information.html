Detailed Description

Convenience struct to hold layer information. Each layer is divided into sectors.

Parameters
perimetersperimeters represented as polylines. Calculated in RPBF_slicer as perimeters only need to be calculated once.
start_vectorvector representing "starting line" for each sector. Used to order perimeters in each island based on circular rotation.
infillgeometric representation of area for infill generation.
start_angleangle used to achieve sector rotation. Used to orient infill.
