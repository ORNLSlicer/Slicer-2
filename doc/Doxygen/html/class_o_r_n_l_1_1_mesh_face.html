Detailed Description

A MeshFace is a 3 dimensional model triangle with 3 points. These points are already converted to integers.

A face has 3 connected faces, corresponding to its 3 edges. Note that a correct model may have more than 2 faces connected via a single edge! In such a case the face_index stored in connected_face_index is the one connected via the outside; see ASCII art below:

: horizontal slice through vertical edge connected to four faces :

* [inside] x|
*          x| <--+--- faces which contain each other in their
* connected_face_index fiels
*    xxxxxxx|   \|/
*    -------+-------
*       ^   |xxxxxxx
*       +-->|x
*       |   |x [inside]
*       |
*     faces which contain each other in their connected_face_index fields
*
