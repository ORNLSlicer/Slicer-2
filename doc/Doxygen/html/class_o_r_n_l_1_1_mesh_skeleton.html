Detailed Description

a class that calculates the mean curvature flow skeleton on a mesh this is done in the following order:

  1. Compute skeleton
  2. Order skeleton points based on Z- > X -> Y height
  3. Extend skeleton to edges of mesh using ray traces
  4. Fetching the approximate distance along a bezier curve fit

void ORNL::MeshSkeleton::order ( )

orders the points of the computed skeleton so a bezier can be calculated

Note
this is done in this order: Z -> X -> Y
Note
To perform a curve fit the points must be in order along the curve. This is done: Compare Z -> Compare X -> Compare Y. If the values are within 10 microns they are considered reasonably close
Epsilon is used to here compare reasonably close values. We use 10 microns here
